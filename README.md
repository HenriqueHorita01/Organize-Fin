# Aplicativo de Organização Financeira Pessoal

Este projeto foi iniciado incrementalmente para gerenciar toda a vida financeira, incluindo receitas, despesas, benefícios, investimentos, patrimônio e orçamentos.

## Estrutura do Projeto

- `docs/prd.md`: Documento de Requisitos do Produto (PRD) consolidando todo o projeto.
- `docs/data_modeling.md`: Modelagem detalhada do banco de dados relacional (entidades, campos e relacionamentos).
- `docs/business_rules.md`: Regras de negócio e comportamento do sistema.
- `docs/functional_architecture.md`: Arquitetura funcional, mapa de telas e fluxos de navegação.
- `docs/ux_product_design.md`: Visão e direcionamento de design de UX/UI (UX sem atrito, fluxo de 3 cliques e psicologia financeira).
- `docs/financial_indicators.md`: Especificação de Indicadores Financeiros e KPIs para o funil de transformação de renda em patrimônio.
- `docs/user_journey.md`: Mapeamento detalhado da jornada de uso (onboarding, uso recorrente e tratamento de cenários especiais).
